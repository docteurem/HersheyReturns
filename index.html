HersheyReturns uses the Hershey fonts dataset to display text with svg characters. The process is similar to https://github.com/techninja/hersheytextjs but is focused on applying hershey fonts to existing html elements. Main features are : uses color, font-size, line-height, letter-spacing, text-align css standard attributes ; independent scale width and scale height with no stroke deformation ; adapts on window resize.
